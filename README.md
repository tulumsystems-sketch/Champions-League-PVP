# Champions League PVP

Plataforma web para jugadores de Free Fire con login, registro, recuperación de contraseña y una base visual para seguir creciendo hacia perfiles, wallet, desafíos y salas.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
